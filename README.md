## Español

**ABYSSEA** es un videojuego de exploración de fondos oceánicos. Desciende al abismo, descubre ecosistemas casi alienígenas y sobrevive en la oscuridad donde ninguna expedición humana ha llegado jamás.

### 🌊 Sobre el Proyecto
Este repositorio contiene la landing page oficial de ABYSSEA, diseñada para ofrecer una experiencia inmersiva mediante efectos parallax, una estética abisal y elementos interactivos.

### 🚀 Características Principales
- **Exploración Profunda:** Experimenta el descenso desde la superficie hasta la Fosa de las Marianas (11.000m) con un medidor de profundidad interactivo.
- **Fauna Abisal:** Descubre y estudia especies únicas basadas en datos oceanográficos reales.
- **Diseño Atmosférico:** Interfaz inmersiva con un motor de burbujas dinámicas en Canvas, efectos bioluminiscentes y parallax multicapa.
- **Mecánicas Interactivas:** Explora las claves del juego (Gestión de Recursos, Catálogo de Especies, Simulación Realista) a través de una interfaz moderna y responsiva.
- **Seguridad de Contenido:** Protección integrada contra copia de contenido no autorizada.

### 🛠️ Tecnologías Utilizadas
- **Estructura:** HTML5 (Semántico)
- **Estilos:** CSS3 (Propiedades personalizadas, Flexbox, Grid, Animaciones)
- **Lógica:** JavaScript Vanilla (ES6+)
- **Recursos:** Iconos SVG personalizados y multimedia optimizada.

### 🔧 Desarrollo
Para ejecutar este proyecto localmente:
1. Clona el repositorio.
2. Abre `index.html` en tu navegador o usa un servidor local como `serve` o `Live Server`.
3. Comandos disponibles:
   - `npm run dev`: Inicia un servidor de desarrollo local.
   - `npm run deploy`: Despliega el proyecto en GitHub Pages.

<a name="english"></a>
## English

**ABYSSEA** is an exploration video game set in the deepest parts of the ocean. Descend into the abyss, discover alien-like ecosystems, and survive in the darkness where no human expedition has ever reached.

### 🌊 About the Project
This repository contains the official landing page for ABYSSEA, designed to provide an immersive experience through parallax effects, deep-sea aesthetics, and interactive elements.

### 🚀 Key Features
- **Deep Exploration:** Experience the descent from the Sunlight Zone to the Challenger Deep (11,000m) with an interactive depth meter.
- **Abyssal Fauna:** Discover and study unique species based on real oceanographic data.
- **Atmospheric Design:** Immersive UI with a custom Canvas-based bubble engine, bioluminescent effects, and multi-layer parallax.
- **Interactive Mechanics:** Explore the game's core features (Resource Management, Species Catalog, Realistic Simulation) through a modern, responsive interface.
- **Content Security:** Integrated protection against unauthorized content copying.

### 🛠️ Tech Stack
- **Structure:** HTML5 (Semantic)
- **Styling:** CSS3 (Custom properties, Flexbox, Grid, Animations)
- **Logic:** Vanilla JavaScript (ES6+)
- **Assets:** Custom SVG icons and optimized media.

### 🔧 Development
To run this project locally:
1. Clone the repository.
2. Open `index.html` in your browser or use a local server like `serve` or `Live Server`.
3. Commands available:
   - `npm run dev`: Starts a local development server.
   - `npm run deploy`: Deploys the project to GitHub Pages.
